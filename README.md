# 💰 FinanceFlow

FinanceFlow é uma aplicação web de controle financeiro pessoal desenvolvida com React, Vite e Firebase.

O objetivo do sistema é permitir que o usuário registre seus gastos do dia a dia e visualize de forma clara para onde seu dinheiro está indo, através de um dashboard simples, rápido e intuitivo.

---

## 📊 Sobre o Projeto

O FinanceFlow foi criado para ser uma alternativa leve e prática a planilhas de controle financeiro.

Ele permite que o usuário acompanhe suas despesas em tempo real, organize gastos por categorias e visualize gráficos que ajudam na análise financeira pessoal.

A interface foi projetada com foco em simplicidade, usabilidade e experiência moderna.

---

## ✨ Funcionalidades

* 🔐 Autenticação de usuários (login e cadastro)
* 💸 Registro de despesas
* ✏️ Edição de gastos
* 🗑️ Exclusão de registros
* 📊 Dashboard com resumo financeiro
* 📈 Gráfico de gastos por categoria
* 🔎 Filtros por período (mês, ano, personalizado)
* ☁️ Armazenamento de dados em nuvem com Firebase
* 📱 Interface responsiva para desktop e mobile

---

## 🧠 Como o sistema funciona

O usuário adiciona seus gastos informando:

* Valor
* Categoria
* Descrição
* Data

Esses dados são armazenados no Firebase Firestore e automaticamente exibidos no dashboard.

O sistema calcula:

* Total gasto
* Distribuição por categoria
* Histórico de despesas

## 📂 Estrutura geral

O projeto é dividido em módulos principais:

* Autenticação de usuários

## 📊 Visualização de dados

O sistema apresenta informações financeiras através de:

* Cards de resumo (total gasto, quantidade de despesas)
* Gráfico de pizza por categoria
* Lista de transações recentes
* Filtros dinâmicos por período

---

## ☁️ Firebase

O Firebase é utilizado como backend principal para:

* Autenticação de usuários
* Armazenamento de dados (Firestore)
* Sincronização em tempo real

---

## 🎯 Objetivo

O objetivo do FinanceFlow é ajudar o usuário a:

* Entender seus hábitos financeiros
* Saber exatamente onde está gastando dinheiro
* Ter mais controle sobre suas finanças pessoais
* Tomar decisões financeiras mais conscientes
